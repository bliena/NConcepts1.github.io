IP Addressing Explanation

  • Purpose: Every device on a network requires a unique identifier; our website uses IP addressing to allow global routers to direct traffic packets to the specific host server.
  • IPv4 vs. IPv6 Resolution: Our deployment demonstrates how a domain name maps to specific numerical addresses, specifically IPv4 (32-bit addresses) or IPv6 (128-bit addresses), ensuring the website is reachable across different network protocols.
  • Static vs. Dynamic Hosting: By using a professional hosting service, our site is assigned to a set of anycast IP addresses, which optimizes the path between the user and the server.
  • Layer 3 Connectivity: In the context of the TCP/IP model, these IP addresses function at the Network layer to handle host-to-host communication.
